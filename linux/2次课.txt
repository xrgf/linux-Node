linux的基本命令：
	命令    已经做好了的应用程序

万物皆文件：  
	都在根目录 /  之下
命令放在这两个地方：
	/bin    /usr/bin

cmd窗口   控制台    命令行窗口
	
	命令是 一行
	命令的标准格式：
		命令名 -选项名 作用域 追加作用域 -追加选项

		作用域 追加作用域 选项 都可以是多个 空格隔开

windows经常出现有空格的文件夹名

当前工作目录：当前在什么位置

pwd：显示当前工作目录的绝对路径

绝对路径：从根目录开始找某个文件的路径
相对路径：从某个文件或者目录开始找某个文件的路径

/  根目录  
~  用户目录   
	linux系统可以有很多个用户
		特殊用户   有且只有一个     root
		其他通通都是普通用户

	root用户   用户目录在   /root    
	普通用户   用户目录在   /home/用户名

创建用户: adduser 
删除用户: userdel
修改密码: passwd 
	passwd 用户名   修改对应用户的密码


root密码：123456

root@qiangsen:~# 

root:当前权限名
@：用来分隔
qiangsen:主机名
::用来分隔
~：当前工作目录名
#  说明当前是特殊用户
$  说明当前是普通用户

shell上 大小写敏感

mkdir:make(造) directory(目录) 
rmdir:remove   	directory
	删除空目录

cd 后面啥都不加  : 切换当前工作目录为 用户主目录

ls 查看当前工作目录下文件
	做任何操作后   接一个 ls

rm 删文件
rm -r 删目录文件

创建普通文件    touch
vim  编辑器   需要自行安装
vi  vim是vi的升级版

3种状态:
	状态1     命令状态
	状态2 	 底行命令状态
	状态3	  编辑状态

命令状态 只能用命令进行操作
编辑状态 只能编辑

进入vi 在 命令状态
	命令状态 ------------->  编辑状态
				a i o
	编辑状态 ------------->  命令状态
				esc键
	命令状态 ------------->  底行命令状态
				:			

:w   保存
:q   退出
:wq  保存后退出  等同于  :x

yy  复制    3yy 复制3行
p   粘贴
dd  剪切     6dd 剪切 6行

重定向:  重新定向
	本来是要到   cmd的   重定向到 文件中  重定向到某个命令里去

输出重定向  >  echo 内容 > 文件 //这是覆盖前面内容的形式 >>是追加到后面内容的形式
输入重定向  <  cat 文件 是读取文件的内容

覆盖重定向 > <
追加重定向 >>  <<

删文件:rm 命令
	sudo rm -rf /*
	-r 删目录
	-f 删之前不提示
	-i 删之前提示

	rm -r -f  a b c
	rm -r   a b c -f
	rm -rf   a b c 
	rm a b c -rf

正则表达式

通配符： *   ？
	*    通配任意个任意字符
	?    通配一个任意字符

文件拷贝: cp     也要带目录   -r选项
	cp -r  多个要拷贝的文件或目录名   目的地目录名
文件移动: mv
	mv 要移动的文件名  移动到的目录名

	如果说 是在当前目录下mv 就是重命名
	mv到其他目录下 才是移动
